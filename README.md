# YOLO-Object-Detection
Object detection using YOLOv8 to detect multiple objects in a single image with bounding boxes in Python.

FEATURES:
 -Objects detection using YOLOv8
 -Works on custom inages (can consists of many objects)
 -Simple Python implementation

REQUIREMENTS:
 -Python 3.8+
 -pip

INSTALLATION:
Install required libraries: pip install ultralytics opency-python

RUN THE PROJECT:
python main.py --source person.jpeg

PROJECT STRUCTURE:
 - main.py-> main detection script
 - person.jpeg-> sample input image
 - requirements.txt-> dependencies

OUTPUT:
 -Detects Objects in the image
 -Displays bounding boxes
